## ✨ Key Features

### 🔐 Authentication

- Secure email/password auth
- Session-based authentication
- Role-based access control (User / Admin)

### 💳 Billing & Subscriptions

- Stripe Checkout integration
- Monthly & yearly plans
- Webhook-driven sync
- Plan-based feature gating

### 📊 SaaS Dashboard

- Real usage metrics
- Subscription status widget
- Upgrade & downgrade flows
- Production UI (not mockups)

### 🛠 Admin Panel

- User management
- Subscription overview
- Revenue insights

## 🏗 Architecture Highlights

- Server-first architecture (Next.js App Router)
- Secure Server Actions
- Event-driven billing system
- Rate-limited APIs
- Production-grade auth flow

Designed to scale **from MVP → real product** without rewrites.
